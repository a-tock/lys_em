lys_em.multislice.getAberrationFunction(sp, tem)[source]

Return the aberration function of the multislice simulation.

The aberration function is calculated from the chi function and the defocus values. The chi function is calculated from the wave number k and the spherical aberration coefficient Cs. The defocus values are given in Angstrom.

Parameters:

  • sp (FunctionSpace) – The function space object.

  • tem (TEM) – The TEM object.

Returns:

A 2D array of shape (len(params), Nx, Ny) where each element is the aberration function at the respective grid point in real space.

Return type:

numpy.ndarray

lys_em.multislice.getChi(sp, tem)[source]

Return the chi function of the multislice simulation.

The chi function is calculated from the wave number k and the spherical aberration coefficient Cs. The wave number k is calculated from the function space object sp and the wavelength lambda of the TEM object tem. The spherical aberration coefficient Cs is given in millimeters.

Parameters:

  • sp (FunctionSpace) – The function space object.

  • tem (TEM) – The TEM object.

Returns:

A function that takes a defocus value df in Angstrom and returns the chi function at that defocus value.

Return type:

callable

lys_em.multislice.getPropagationTerm(sp, tem)[source]

Return the propagation term of the wave transfer function.

The propagation term is calculated from the wave number k and the propagation distance dz. The wave number k is calculated from the crystal structure and the wavelength lamb. The propagation distance dz is given in Angstrom.

The wave number k is represented as a 2D array of shape (Nx, Ny) where each element is the wave number at the respective grid point in reciprocal space. The unit of k is rad/A.

The propagation term is calculated as exp(1j * k * dz).

Parameters:

  • sp (FunctionSpace) – The function space object.

  • tem (TEM) – The TEM object.

Returns:

A 2D array of shape (Nx, Ny) where each element is the propagation term at the respective grid point in reciprocal space.

Return type:

numpy.ndarray
